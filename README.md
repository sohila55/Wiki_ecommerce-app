📱 About The App
A complete e-commerce mobile application built with Flutter, featuring a clean and intuitive shopping experience. The app uses Firebase as a real backend for authentication and data storage, and Cubit for smooth and predictable state management.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------
✨ Features

🔐 User Authentication — Register & Login with Firebase Auth
🏠 Home Screen — Browse all products in a clean layout
🗂️ Categories — Filter products by category
🔍 Product Details — Full product info with images and description
🛒 Cart — Add, remove, and manage items before checkout
❤️ Favourites / Wishlist — Save favourite products for later
👤 Profile — View and manage user information
📲 Bottom Navigation Bar — Smooth navigation between main screens

🚀 Getting Started

### Prerequisites

Flutter SDK installed
Firebase project set up

### Installation

# Clone the repo
git clone https://github.com/sohila55/YOUR_REPO_NAME.git

# Install dependencies

flutter pub get

# Connect your Firebase project

Add your google-services.json to android/app/
Add your GoogleService-Info.plist to ios/Runner/


# Run the app
  
  flutter run
